# Система выявления опасного контента в текстах

Классификатор текста на базе ruBERT. Определяет 5 классов: насилие, ненависть, суицидальный контент, дезинформацию и нейтральный текст.

## Файлы проекта

**`content_analyzer_gui.py`** — исходный код GUI-приложения.

**`learning_ruBert.ipynb`** — обучение модели ruBERT на датасете.

---

## Быстрый старт

### 1. Скачать архив

Скачайте zip-архив с программой и моделью по ссылке:
[Google Drive](https://drive.google.com/drive/folders/1ErRJQ44-nKLFUSea4sO1EIPVmO9iMEa7?usp=drive_link)

Архив содержит:
- папку `Content_analyzer` — скомпилированная программа
- папку `rubert_toxic_classifier_v3` — обученная модель

### 2. Распаковать архив

Распакуйте скачанный zip в любую удобную папку. Важно, чтобы путь к папке не содержал кириллицы — это может вызвать ошибки при запуске.

### 3. Запустить программу

Откройте файл:

```
Content_analyzer\dist\ContentAnalyzer\ContentAnalyzer.exe
```

> Не переносите `ContentAnalyzer.exe` отдельно от папки — программа не запустится без сопутствующих файлов.

---

## Работа с программой

1. При первом запуске нажмите **«Загрузить модель»** и укажите путь к папке `rubert_toxic_classifier_v3`.
2. После загрузки модели кнопка **«Анализировать»** станет активной.
3. Введите текст вручную или загрузите `.txt`-файл через **«Загрузить из файла»**.
4. Нажмите **«Анализировать»** — результат появится справа.

В результате отображается:
- класс текста и уверенность модели
- вероятности по всем пяти классам (если включена соответствующая опция в настройках)

Результаты можно сохранить в JSON или скопировать в буфер обмена. История анализов доступна на вкладке **«История»** — её можно экспортировать в JSON или CSV.
